# [!DNL Quality Patches Tool] : outil en libre-service pour les correctifs de qualité

[!DNL Quality Patches Tool] vous permet d’appliquer, d’annuler et d’afficher des informations générales sur les correctifs de qualité disponibles pour votre version installée d’Adobe Commerce (sur site et sur l’infrastructure cloud) ou de Magento Open Source.

Cet outil permet de résoudre plus rapidement les problèmes en libre-service. Vous pouvez ainsi appliquer facilement les correctifs recommandés par le support Adobe Commerce pour résoudre les problèmes susceptibles de se produire avec Adobe Commerce.

>[!NOTE]
>
>Vous devez utiliser l’outil pour extraire les correctifs de qualité nécessaires au cas par cas uniquement et non dans le cadre d’une approche à long terme de la maintenance du code Adobe Commerce et Magento Open Source. La mise à niveau vers la dernière version d’Adobe Commerce ou de Magento Open Source reste la meilleure méthode pour résoudre les problèmes de qualité et doit être prioritaire.

## Obtenir et utiliser le [!DNL Quality Patches Tool]

L’outil est disponible sous la forme d’un package de compositeur. Pour Adobe Commerce sur les infrastructures en nuage, le module est inclus dans le module ECE-Tools.

Une fois l’outil installé, vous pouvez vérifier les correctifs disponibles et les appliquer ou les rétablir à tout moment.
