# ACP2E-4401 : la [!UICONTROL Scheduled Update Preview] de la page d’accueil avec des produits configurables redirige vers la page de maintenance.

Le correctif ACP2E-4401 corrige le problème en raison duquel la **[!UICONTROL Scheduled Update Preview]** du [!UICONTROL Home Page] avec des produits configurables redirige vers la **page de maintenance** ou affiche une erreur au lieu d’afficher la liste des produits. Ce correctif est disponible lorsque la version 1.1.81 de [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) est installée. L’ID du correctif est ACP2E-4401. Notez que ce problème doit être résolu dans Adobe Commerce 2.4.9.

## Produits et versions concernés

**Le correctif est créé pour la version Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8-p3

**Compatible avec les versions d’Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8 - 2.4.8-p5

>[!NOTE]
>
>Le correctif peut s’appliquer à d’autres versions avec de nouvelles versions de [!DNL Quality Patches Tool]. Pour vérifier si le correctif est compatible avec votre version d’Adobe Commerce, mettez à jour le package `magento/quality-patches` vers la dernière version et vérifiez la compatibilité sur la page [[!DNL Quality Patches Tool] : Rechercher des correctifs](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilisez l’ID du correctif comme mot-clé de recherche pour localiser le correctif.

## Problème

La **[!UICONTROL Scheduled Update Preview]** de l’[!UICONTROL Home Page] avec des produits configurables redirige vers la **page de maintenance** ou affiche une erreur au lieu d’afficher la liste des produits.

<u>Procédure à suivre </u> :

1. Créez un produit configurable avec plusieurs options et affectez-le à une catégorie.
1. Dans la barre latérale d’administration, accédez à **Contenu** > **[!UICONTROL Elements]** > **[!UICONTROL Pages]**, puis ouvrez la page CMS affectée en tant que **Page d’accueil**.
1. Dans la zone de **[!UICONTROL Scheduled Changes]**, cliquez sur **[!UICONTROL Schedule New Update]**, créez une mise à jour planifiée pour la page, puis enregistrez la mise à jour.
1. Dans la zone de **[!UICONTROL Scheduled Changes]**, cliquez sur **[!UICONTROL Preview]**.
1. Dans la fenêtre d&#39;aperçu, cliquez sur un lien de catégorie.

<u>Résultats attendus</u> :

1. L’aperçu de l’évaluation s’affiche correctement sans erreur.
1. Le lien de catégorie ouvre la page de liste des produits comme prévu.

<u>Résultats réels</u> :

1. La navigation dans l’aperçu ouvre la **Page de maintenance** ou déclenche une erreur irrécupérable.
1. Le fichier `var/log/exception.log` contient une erreur similaire à *`report.CRITICAL: TypeError: Magento\Weee\Plugin\ConfigurableProduct\Pricing\FinalPriceResolver::afterResolvePrice(): Return value must be of type float, string returned in /var/app/code/Magento/Weee/Plugin/ConfigurableProduct/Pricing/FinalPriceResolver.php:51`*.
