# ACSD-69331 : la création du dossier Galerie de médias échoue pour les utilisateurs administrateurs disposant uniquement d’une autorisation **[!UICONTROL Create Folder]**

Le correctif ACSD-69331 corrige le problème en raison duquel les utilisateurs administrateurs ne pouvaient pas créer de dossiers dans la Galerie de médias CMS lorsque seule l’autorisation **[!UICONTROL Create Folder]** était accordée. Après le correctif, ils peuvent créer des dossiers comme prévu. Ce correctif est disponible lorsque la version 1.1.76 de [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) est installée. L’ID du correctif est ACSD-69331. Notez que ce problème doit être résolu dans Adobe Commerce 2.4.9.

## Produits et versions concernés

**Le correctif est créé pour la version Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.7-p8

**Compatible avec les versions d’Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.5 - 2.4.8-p3

>[!NOTE]
>
>Le correctif peut s’appliquer à d’autres versions avec de nouvelles versions de [!DNL Quality Patches Tool]. Pour vérifier si le correctif est compatible avec votre version d’Adobe Commerce, mettez à jour le package `magento/quality-patches` vers la dernière version et vérifiez la compatibilité sur la page [[!DNL Quality Patches Tool] : Rechercher des correctifs](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilisez l’ID du correctif comme mot-clé de recherche pour localiser le correctif.

## Problème

Les utilisateurs administrateurs disposant uniquement de l’autorisation **[!UICONTROL Create Folder]** dans la Galerie de médias ne peuvent pas créer de dossiers, sauf si l’autorisation **[!UICONTROL Delete Folder]** est également accordée.

<u>Procédure à suivre </u> :

1. Connectez-vous au panneau d’administration.
1. Créez un nouveau rôle avec un accès à **[!UICONTROL Dashboard]** et **[!UICONTROL Content]** > **[!UICONTROL Elements]** > **[!UICONTROL Media Gallery]**, en accordant uniquement l’autorisation **[!UICONTROL Create Folder]** (n’accordez pas l’autorisation **[!UICONTROL Delete Folder]**).
1. Créez un utilisateur administrateur et attribuez-lui ce rôle.
1. Connectez-vous en tant que nouvel utilisateur administrateur.
1. Accédez à **[!UICONTROL Content]** > *[!UICONTROL Media]* > **[!UICONTROL Media Gallery]**.
1. Sélectionnez un répertoire (par exemple, `catalog`).
1. Cliquez sur **[!UICONTROL Create Folder]**.
1. Saisissez et confirmez un nom de dossier.

<u>Résultats attendus</u> :

* La création de dossiers fonctionne pour les utilisateurs administrateurs uniquement avec l’autorisation **[!UICONTROL Create Folder]**.
* L’autorisation **[!UICONTROL Delete Folder]** n’est pas requise pour créer des dossiers.

<u>Résultats réels</u> :

* L’utilisateur ne peut pas créer de dossiers avec uniquement l’autorisation **[!UICONTROL Create Folder]**.
* Pour créer des dossiers, il est nécessaire d’accorder l’autorisation **[!UICONTROL Delete Folder]**.
