# ACP2E-4601 : Réenregistrer un `parent_id` d&#39;auto-référence de transaction d&#39;autorisation de paiement et déclencher une boucle infinie/OOM

Le correctif ACP2E-4601 corrige le problème de comportement inefficace du traitement des transactions de paiement lors de la réenregistrement des transactions d’autorisation sans `parent_txn_id`. Ce correctif est disponible lorsque la version 1.1.79 de [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) est installée. L’ID du correctif est ACP2E-4601. Notez que ce problème doit être résolu dans Adobe Commerce 2.4.9.

## Produits et versions concernés

**Le correctif est créé pour la version Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8-p3

**Compatible avec les versions d’Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8 - 2.4.8-p4

>[!NOTE]
>
>Le correctif peut s’appliquer à d’autres versions avec de nouvelles versions de [!DNL Quality Patches Tool]. Pour vérifier si le correctif est compatible avec votre version d’Adobe Commerce, mettez à jour le package `magento/quality-patches` vers la dernière version et vérifiez la compatibilité sur la page [[!DNL Quality Patches Tool] : Rechercher des correctifs](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilisez l’ID du correctif comme mot-clé de recherche pour localiser le correctif.

## Problème

Le traitement des transactions de paiement se comporte de manière inefficace lors de la réenregistrement des transactions d&#39;autorisation créées sans `parent_txn_id`.

<u>Procédure à suivre </u> :

1. Configurez un mode de paiement en ligne.
1. Passer une commande qui crée une transaction de paiement d&#39;autorisation sans `parent_txn_id`.
1. Réenregistrez la transaction d’autorisation.

<u>Résultats attendus</u> :

Le traitement des transactions de paiement s&#39;est terminé avec succès.

<u>Résultats réels</u> :

La réenregistrement de la transaction d’autorisation échoue.
