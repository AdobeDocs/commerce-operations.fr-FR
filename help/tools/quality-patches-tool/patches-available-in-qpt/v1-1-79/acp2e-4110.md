# ACP2E-4110 : Les prix groupés des produits avec un **[!UICONTROL Special Price]** différent entre PDP/PLP et le panier dans une devise autre que celle par défaut

Le correctif ACP2E-4110 corrige le problème où les lots de produits avec un **[!UICONTROL Special Price]** affichent des montants incorrects sur la page des détails du produit (PDP) et la page de liste des produits (PLP) dans une devise autre que celle par défaut. Ce correctif est disponible lorsque la version 1.1.79 de [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) est installée. L’ID du correctif est ACP2E-4110. Notez que ce problème doit être résolu dans Adobe Commerce 2.4.9.

## Produits et versions concernés

**Le correctif est créé pour la version Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8-p3

**Compatible avec les versions d’Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.7 - 2.4.8-p4

>[!NOTE]
>
>Le correctif peut s’appliquer à d’autres versions avec de nouvelles versions de [!DNL Quality Patches Tool]. Pour vérifier si le correctif est compatible avec votre version d’Adobe Commerce, mettez à jour le package `magento/quality-patches` vers la dernière version et vérifiez la compatibilité sur la page [[!DNL Quality Patches Tool] : Rechercher des correctifs](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilisez l’ID du correctif comme mot-clé de recherche pour localiser le correctif.

## Problème

Les produits groupés avec un **[!UICONTROL Special Price]** affichent des montants incorrects sur le PDP et le PLP dans une devise autre que celle par défaut.

<u>Procédure à suivre </u> :

1. Créez un produit simple. Définissez la **[!UICONTROL Price]** = *40,25*.
1. Créez un produit groupé.
1. Définissez la **[!UICONTROL Special Price]** pour le produit groupé = 97,5 %**&#x200B; et la &#x200B;** [!UICONTROL Price View]** = *Plage de prix*.
1. Attribuez le produit simple en tant qu’option du produit groupé. Définissez la **[!UICONTROL Default Quantity]** = *8*.
1. Affectez le produit groupé à une catégorie visible afin qu’il puisse être vérifié sur la page PLP/catégorie.
1. Conservez la source de tarification en *USD*, définissez le **[!UICONTROL Allowed Currency]** = *GBP* et définissez le **[!UICONTROL Default Display Currency]** par défaut = *GBP* pour le storefront.
1. Définissez le taux de conversion de devise pour USD sur GBP = *0,75*.
1. Effacez le cache et exécutez la réindexation.
1. Ouvrez le storefront dans la vue GBP et notez le prix du bundle sur la page PLP/category et le PDP.
1. Ajoutez le produit groupé au panier.
1. Comparez le prix affiché dans le panier avec le prix sur le PLP/PDP.

<u>Résultats attendus</u> :

Le prix du bundle du produit doit être le même sur le PLP, PDP et le panier dans une devise autre que celle par défaut.

<u>Résultats réels</u> :

Le prix affiché diffère entre la page/la liste de produits et le panier.
Exemple :
Prix sur la page produit/PDP/PLP = *235,52 GBP*
Prix dans le panier = 235,44 *GBP*
