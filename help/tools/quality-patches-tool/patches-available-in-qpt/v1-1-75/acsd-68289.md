# ACSD-68289 : la recherche en texte intégral ne renvoie aucun résultat lorsqu’une correspondance minimale s’applique à tous les champs pouvant faire l’objet de recherches

Le correctif ACSD-68289 corrige le problème où la recherche en texte intégral renvoie des produits lorsque la condition de correspondance minimale est remplie pour tous les champs pouvant faire l’objet d’une recherche, plutôt que d’avoir besoin d’un seul champ pour satisfaire tous les termes de recherche. Ce correctif est disponible lorsque la version 1.1.75 de [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) est installée. L’ID du correctif est ACSD-68289. Notez que ce problème doit être résolu dans Adobe Commerce 2.4.9.

## Produits et versions concernés

**Le correctif est créé pour la version Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8-p2

**Compatible avec les versions d’Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>Le correctif peut s’appliquer à d’autres versions avec de nouvelles versions de [!DNL Quality Patches Tool]. Pour vérifier si le correctif est compatible avec votre version d’Adobe Commerce, mettez à jour le package `magento/quality-patches` vers la dernière version et vérifiez la compatibilité sur la page [[!DNL Quality Patches Tool] : Rechercher des correctifs](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilisez l’ID du correctif comme mot-clé de recherche pour localiser le correctif.

## Problème

La recherche en texte intégral renvoie des produits lorsque l’exigence de correspondance minimale est satisfaite pour plusieurs champs, au lieu d’exiger que tous les termes correspondent dans un seul champ.

<u>Procédure à suivre </u> :

1. Créez un attribut de produit :
   * **[!UICONTROL Default Label]** : *Année du modèle*
   * **[!UICONTROL Catalog Input Type for Store Owner]** : *Champ de texte*
   * **[!UICONTROL Use In Search]** : *Oui*
1. Ajoutez l’attribut au jeu d’attributs par défaut.
1. Créez un produit :
   * **[!UICONTROL Product Name]** : *Honda Jazz GDI*
   * **[!UICONTROL Model Year]** : *2008*
1. Dans le panneau **[!UICONTROL Admin]**, accédez à **[!UICONTROL Stores]** > **[!UICONTROL Settings]** > **[!UICONTROL Configuration]** > **[!UICONTROL Catalog]** > **[!UICONTROL Catalog Search]**. Définissez **[!UICONTROL Minimum Terms to Match]** sous Paramètres de recherche sur *3&lt;-1 5&lt;50 %*.
1. Exécutez la réindexation complète.
1. Dans la vitrine, recherchez *2008 honda*.

<u>Résultat attendu </u> :

Le produit créé apparaît dans les résultats de la recherche.

<u>Résultat réel</u> :

Aucun résultat de recherche renvoyé.
