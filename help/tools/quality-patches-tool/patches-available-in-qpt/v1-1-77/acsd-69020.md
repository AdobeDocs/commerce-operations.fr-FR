# ACSD-69020 : les produits configurables apparaissent dans [!DNL Page Builder] carrousels lorsque les produits enfants correspondent aux filtres

Le correctif ACSD-69020 corrige le problème où un produit configurable est automatiquement inclus dans [!DNL Page Builder] listes de carrousel de produits lorsqu’un produit enfant remplit les conditions de filtrage. Le produit parent est inclus uniquement si le produit enfant correspondant n&#39;est pas visible individuellement. Ce correctif est disponible lorsque la version 1.1.77 de [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) est installée. L’ID du correctif est ACSD-69020. Notez que ce problème doit être résolu dans Adobe Commerce 2.4.9.

## Produits et versions concernés

**Le correctif est créé pour la version Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.7-p7

**Compatible avec les versions d’Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.7 - 2.4.7-p8

>[!NOTE]
>
>Le correctif peut s’appliquer à d’autres versions avec de nouvelles versions de [!DNL Quality Patches Tool]. Pour vérifier si le correctif est compatible avec votre version d’Adobe Commerce, mettez à jour le package `magento/quality-patches` vers la dernière version et vérifiez la compatibilité sur la page [[!DNL Quality Patches Tool] : Rechercher des correctifs](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=fr). Utilisez l’ID du correctif comme mot-clé de recherche pour localiser le correctif.

## Problème

Un problème se produit lorsqu’un produit configurable est automatiquement inclus dans [!DNL Page Builder] listes de carrousel de produits lorsque l’un de ses produits enfants remplit les conditions de filtrage. Le produit parent est inclus uniquement si le produit enfant correspondant n&#39;est pas visible individuellement.

<u>Procédure à suivre </u> :

1. Créez une mise à jour d’évaluation du contenu pour un produit existant, attendez le début de la mise à jour, puis exécutez les tâches cron.
1. Créez les produits suivants dans l’ordre :
   1. Créez un `testconfig` de produit configurable avec trois produits enfants :
      * `testconfig-option1`
      * `testconfig-option2`
      * `testconfig-option3`
   1. Créez trois produits simples :
      * `testoption1`
      * `testoption2`
      * `testoption3`
   1. Créez un `testconfigtwo` de produit configurable avec trois produits enfants.
   1. Définissez **[!UICONTROL Visibility]** sur *Catalogue* pour tous les produits simples (enfants).
1. Exécutez les traitements de réindexation ou cron.
1. Échangez les produits enfants entre les produits configurables :
   * Modifiez le `testconfig` et supprimez le `testconfig-option1` de produit enfant.
   * Modifiez le `testconfigtwo` et supprimez le `testconfigtwo-option3` de produit enfant.
   * Modifiez le `testconfig`, ajoutez le `testconfigtwo-option1` de produit enfant, puis cliquez sur **[!UICONTROL Save]**.
   * Modifiez le `testconfigtwo`, ajoutez le `testconfig-option3` de produit enfant, puis cliquez sur **[!UICONTROL Save]**.
1. Exécutez à nouveau les traitements de réindexation ou cron.
1. Dans le panneau **[!UICONTROL Admin]**, accédez à **[!UICONTROL Content]** > **[!UICONTROL Pages]** > **[!UICONTROL Edit any page]**.
1. Sous **[!UICONTROL Content]** section, cliquez sur **[!UICONTROL Edit with Page Builder]**.
1. Ajouter une nouvelle ligne
1. Ajouter ou modifier du contenu Produits :
   1. Définissez **[!UICONTROL Select Products By]** sur *SKU*.
   1. Définissez **[!UICONTROL SKU]** sur `testconfigtwo-option1`, puis cliquez sur **[!UICONTROL Save]**.
1. Observez les produits configurables affichés dans le carrousel de produits.

<u>Résultats attendus</u> :

Seuls les produits configurables dont les produits enfants affectés ne sont pas visibles individuellement apparaissent dans le carrousel de produit.

<u>Résultats réels</u> :

Les produits configurables auxquels des `testconfigtwo-option1` de produit enfant sont affectés apparaissent dans le carrousel de produit.
