# ACP2E-4555 : Les numéros de téléphone contenant des points ou des barres obliques ne sont pas validés correctement

Le correctif ACP2E-4555 corrige le problème de non-validation des numéros de téléphone modernes contenant des `.` ou des `/`. Ce correctif est disponible lorsque la version 1.1.78 de [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) est installée. L’ID du correctif est ACP2E-4555. Notez que ce problème doit être résolu dans Adobe Commerce 2.4.9.

## Produits et versions concernés

**Le correctif est créé pour la version Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8-p3

**Compatible avec les versions d’Adobe Commerce :**

* Adobe Commerce (toutes les méthodes de déploiement) 2.4.8 - 2.4.8-p4

>[!NOTE]
>
>Le correctif peut s’appliquer à d’autres versions avec de nouvelles versions de [!DNL Quality Patches Tool]. Pour vérifier si le correctif est compatible avec votre version d’Adobe Commerce, mettez à jour le package `magento/quality-patches` vers la dernière version et vérifiez la compatibilité sur la page [[!DNL Quality Patches Tool] : Rechercher des correctifs](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilisez l’ID du correctif comme mot-clé de recherche pour localiser le correctif.

## Problème

Les numéros de téléphone modernes contenant `.` ou `/` ne sont pas validés.

<u>Procédure à suivre </u> :

1. Connectez-vous en tant que client sur le storefront.
1. Ajoutez une nouvelle adresse au carnet d’adresses avec un numéro de téléphone, par exemple : `06.76.40.32.22` ou `+43680/2149568`.
1. Enregistrez les détails de l’adresse.

<u>Résultats attendus</u> :

Il est possible d&#39;utiliser des formats téléphoniques modernes avec des points et des barres obliques.

<u>Résultats réels</u> :

La validation échoue pour le champ téléphonique en raison des caractères `.` ou `/`.

## Application du correctif

Pour appliquer des correctifs individuels, utilisez les liens suivants en fonction de votre méthode de déploiement :
